### Angular Task

----

`Angular 4.x` 搭建的一个企业级协作平台

如果 `npm/yarn install` 安装完成后运行报 `materials` 相关错误，则可以尝试手动进行安装 `materials`，版本为 `@2.0.0-beta.7`

```
$ npm/yarn install --save @angular/material@2.0.0-beta.7
```
